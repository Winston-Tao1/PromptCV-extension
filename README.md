## 🎯 产品定位

**PromptCV** 是一款专为AI工作者设计的Chrome浏览器扩展，致力于解决提示词管理混乱、重复输入、多模型切换繁琐等痛点。

### 为什么选择PromptCV？

- 📚 **集中管理** - 告别散落在各处的提示词，一站式管理
- ⚡ **极速调用** - 一键复制，无需重复输入
- 💾 **本地缓存** - 支持文件、图片、富文本，安全私密
- 🎯 **多模型支持** - 轻松管理多个AI模型配置
- 🔄 **云同步** - 提示词跨设备同步，随时随地访问

## 🚀 快速开始

### 方式一：开发者模式安装（立即使用）

1. **下载项目**
   ```bash
   git clone https://github.com/Winston-Tao1/PromptCV-extension.git
   cd PromptCV-extension
   ```

2. **生成图标**
   - 打开 `generate-icons.html`
   - 点击"下载全部图标"
   - 将4个PNG文件放入 `icons/` 文件夹

3. **加载到Chrome**
   - 访问 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目文件夹

4. **完成！** 🎉
   - 工具栏出现PromptCV图标
   - 点击开始使用

## 📖 使用指南

### 1️⃣ 模型配置

#### 添加模型
1. 点击右上角"⚙️ 设置"
2. 填写API信息（从胜算云平台获取）
3. 点击"保存配置"
4. 点击灰色开关激活模型

#### 管理模型
- **启动** → 点击灰色开关（变绿）
- **关闭** → 点击绿色开关（变灰）
- **删除** → 点击"删除"按钮并确认

### 2️⃣ 提示词管理

#### 添加提示词
1. 点击"**添加**"标签
2. 选择AI平台
3. 输入提示词内容
4. 点击"保存"（自动复制）

#### 使用提示词
- **收藏** → 点击⭐图标
- **复制** → 点击"📋 复制"按钮
- **编辑** → 点击卡片任意位置
- **删除** → 右键菜单或编辑器内删除

### 3️⃣ 缓存功能

#### 编辑文本
1. 点击"**缓存**"标签
2. 使用工具栏格式化文本
3. 内容自动保存

#### 上传文件
1. 点击"📤 上传文件"
2. 选择任意文件（支持多选）
3. 文件插入光标位置

#### 粘贴图片
1. 复制图片（截图/复制图片）
2. 在编辑器按 Ctrl+V
3. 图片自动插入

## 🐛 常见问题

### Q: 数据会丢失吗？
**A:** 不会！
- 提示词：云同步，换设备不丢失
- 缓存：本地存储，清除浏览器才丢失
- 建议：重要数据定期备份

### Q: 支持哪些文件格式？
**A:** 所有格式！
- 文档、表格、演示
- 图片、音视频
- 代码、压缩包
- 无限制！

### Q: 如何获取API Key？
**A:** 推荐胜算云平台
- 访问 shengsuanyun.com
- 注册账号
- 在控制台获取API Key

### Q: 模型配置无法关闭？
**A:** 请更新到v2.1.0+
- 绿色开关点击变灰
- 状态切换明显
